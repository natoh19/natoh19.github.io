Recent Projects

Full Stack Web Apps A Selection of Recent Projects

Sophora Repo

A Sephora inspired beauty products e-commerce application built with a React, Redux frontend and a Flask/Python backend. Users can browse, add items to their cart, adjust quantity, remove products from the cart and checkout. Users can also create wishlists by liking products which are saved to their My Account page where they can be removed.

ThrifTEE Repo

A ThredUp inspired e-commerce application built with a React, Redux frontend and a Flask/Python backend. Users can browse by category, add items to their cart and search the site by keyword. Using Redux in conjunction with local storage the contents of the user’s cart persists in between site visits.

SoundOm Repo

SoundOm, a SoundCloud inspired web application built with a React, Redux frontend and an Express backend. Registered users can upload tracks to AWS, remove tracks they’ve uploaded and view their list of uploaded tracks from their Profile page. All users can listen to tracks, pause and skip to next and previous tracks from the Home page.
